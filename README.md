
# Advanced Data Mining Project – Deliverable 1

## Dataset
Customer Churn Dataset from Kaggle containing 7043 records and 21 attributes.

## Objective
To perform data collection, cleaning, and exploratory data analysis to prepare the dataset for predictive modeling.

## Data Cleaning Steps
- Checked missing values
- Removed duplicates
- Converted incorrect data types
- Identified outliers

## Exploratory Analysis
Visualizations used:
- Histogram
- Count Plot
- Correlation Heatmap
- Box Plot

## Key Insights
- Customer churn patterns observed
- Tenure and monthly charges influence churn
- Some outliers present in billing features

## Challenges
- Handling missing values
- Converting inconsistent data types
